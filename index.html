TuringModels

StatisticalRethinkingJulia/TuringModels.jl

This site shows the Julia versions of the Bayesian models described in Statistical Rethinking Edition 1 (McElreath, 2015) and 2 (McElreath, 2020). The models are listed at Models.

We have tried our best to let this site be without mistakes. However, if you find a mistake on this website, please let us know. The code for this website is available on GitHub.

Models

The models are listed below. Each page aims to contain all the code required to reproduce the results. In other words, it should be possible to get the same output by just copying the code and the accompanying dataset. Furthermore, we try to stick to Julia styling conventions where possible. Therefore, we use unicode symbols in the models. For example, where the book lists alpha, we will use Î± because the Julia language allows that. As another example, we call DataFrame variables df, which is also the convention in R and Python.

Before you look at the models below, you might want to look at the Basic Example.
